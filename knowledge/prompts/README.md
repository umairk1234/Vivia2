## How to Use

### In Cursor/Claude:

```
Read the prompt at knowledge/prompts/[prompt-name].md and use it to analyze [input].
```

### Example:

```
Read the prompt at knowledge/prompts/competitor-lp-breakdown.md 
and use it to analyze this screenshot: [paste screenshot]
```

---

## Adding New Prompts

When creating a new prompt, include:

1. **Purpose** - What the prompt does
2. **System Prompt** - The AI instruction
3. **Instructions** - Step-by-step format guide
4. **Examples** - Sample inputs/outputs
5. **Version History** - Track changes
